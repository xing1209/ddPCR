# SMDetect

## 功能描述

SMDetect是一款面向POCT（即时检验）场景的智能荧光分析系统，适用于微流控领域，能够高效处理荧光目标检测与计数任务。具备：
- 三合一检测模式：静态图片（任意分辨率）/视频流/实时显微影像
- 双引擎架构：YOLOv5检测引擎（目标识别）+DeepSORT追踪引擎（动态计数）
